thfSearchPhrases

Syntax

#include <trulyhandsfree.h>
int thfSearchPhrases(thf_t **thf, searchs_t *sch, char ***map, int *count);

Description

Returns an array of trigger phrases used in a search.
The array index is the trigger phrase ID and the final entry in the returned array is NULL.
Use thfFree to release the memory allocated for the returned phrase array.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
searchs_t * sch in Search object.
char *** map out Address of phrase array, allocated by this function.
int * count out If non-NULL the number of phrases is stored in count.

Return

Returns 1 to indicate success and 0 for an error. Use thfGetLastError to get error message.

Errors

String Description
search object not found sch parameter is NULL.
search not a grammar or spotting variety sch parameter is of a type not supported by this function.
