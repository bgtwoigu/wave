How to Perform Gender Identification

Gender identification is similar to speaker identification, except that the only two identities are 'female' and 'male', and enrollment and training a speaker is not required.

Step 1: Read in a Gender Model

An existing gender model is read from file into the recognizer object using thfSpeakerReadGenderModel. Gender models are currently specific to a target phrase. Please contact Sensory for development of a phrase-specific gender model.

Step 2: Perform Gender Identification

Gender identification is performed after a call to thfRecogResult returns a value indicating that the target phrase has been recognized. Identification is done by calling thfSpeakerGender, which returns a gender score between 0.0 and 1.0. A value less than 0.5 indicates a male speaker, and a value greater than or equal to 0.5 indicates a female speaker.
