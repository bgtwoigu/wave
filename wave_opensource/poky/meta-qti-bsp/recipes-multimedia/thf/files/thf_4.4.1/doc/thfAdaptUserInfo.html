thfAdaptUserInfo

Syntax

#include <trulyhandsfree.h>
int thfAdaptUserInfo(thf_t **thf, adaptObj_t *aObj, char *taskName, 
                     thfAdaptUserInfo_t **out);

Description

Return an object containing information about all users in a task. This object is of type thfAdaptUserInfo_t and information can be obtained by examining the values in this structure. When this object is no longer needed, it must be destroyed with a call to thfAdaptUserInfoDestroy.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
adaptObj_t * aObj in adapt object.
char * taskName in name of current task.
thfAdaptUserInfo_t ** out userInfoOut user information object.

Return

Returns 1 on success or 0 on error.

Errors

String Description
adapt object not found A NULL pointer was passed in for the adapt object; pass in a correctly-initialized adapt object pointer created by thfAdaptCreate.
adapt object not initialized A non-NULL pointer was passed in for the adapt object, but this pointer does not contain a properly-initialized adapt object that is the output of thfAdaptCreate.
task name not specified A valid task name must be provided; a task with this name must have been created with a previous call to thfAdaptTaskCreate.
internal error getting enroll info There was an internal error. Please contact customer support.
internal error matching numEnroll There was an internal error. Please contact customer support.
