thfSpeakerStoreLastRecording

Syntax

#include <trulyhandsfree.h>
int thfSpeakerStoreLastRecording(thf_t **thf, recog_t *r, unsigned short sID, 
                                 short score, short force);

Description

Takes the last utterance identified by thfRecogResult and stores it internally as a recording spoken by the specified speaker.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
unsigned short sID in The identification (integer value) of the speaker who uttered the last trigger identified by thfRecogResult.
short score in A score that indicates the quality of the recording. Typically, this is the confidence score returned by thfRecogResult multiplied by some large constant in order to convert from floating-point to integer.
short force in A flag that indicates whether this recording must be stored (potentially overwriting an earlier recording of this user) or not. If 'force' is 1, then this recording will always be stored, regardless of the 'score' value. If 'force' is 0 and this function has been called more times than the number of desired recordings (specified in thfSpeakerInit, then this recording will only be stored if it has a 'score' value greater than or equal to at least one of the previously-stored recordings. In this case, the recording with the lowest score will be replaced. If two recordings have the same score, the more recent recording is considered better.

Return

Returns 1 on success or 0 on error.

Errors

String Description
recog object not found Invalid recognizer object handle.
speaker object not yet initialized Invalid speaker object handle.
internal error: failed to store last recording Unexpected internal error. Contact customer support.
