thfPhonemeSearchCreateFromFile

Syntax

#include <trulyhandsfree.h>
phsearch_t *thfPhonemeSearchCreateFromFile(thf_t **thf, recog_t *r, 
                                           const char *filename, 
                                           unsigned short nbest);

Description

Create a phoneme-level search using data from file, and associate it with the specified recognizer.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
const char * filename in name of file containing phoneme search data.
unsigned short nbest in number of results to return. A typical value is 1.

Return

Returns a pointer to a phsearch_t object on success or NULL on error. Invalid recognizer object handle.

Errors

String Description
Recog object incompatible The specified recognizer uses 'hybrid' models, which is not allowed with phoneme-level searches.
Unsupported search type The data in the specified file is not consistent with a phoneme-level search.
internal error: Incomplete search object Internal error: Contact customer support
internal error: During search object creation Internal error: Contact customer support
internal error: Unsupported search type Internal error: Contact customer support
