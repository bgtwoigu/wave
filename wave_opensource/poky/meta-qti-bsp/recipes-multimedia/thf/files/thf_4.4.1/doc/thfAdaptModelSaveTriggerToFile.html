thfAdaptModelSaveTriggerToFile

Syntax

#include <trulyhandsfree.h>
int thfAdaptModelSaveTriggerToFile(thf_t **thf, adaptObj_t *aObj, 
                                   thfModelTrigger_t *modelTrigger, 
                                   char *searchFile, char *recogFile);

Description

Save the speaker-adapted trigger to file. The resulting recognizer file and search file can be read with calls to thfRecogCreateFromFile and thfSearchCreateFromFile.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
adaptObj_t * aObj in adapt object.
thfModelTrigger_t * modelTrigger in trigger created by thfAdaptModelGetTrigger.
char * searchFile in name of search file to be created.
char * recogFile in name of recognizer file to be created.

Return

Returns 1 on success or 0 on error.

Errors

String Description
adapt object not found A NULL pointer was passed in for the adapt object; pass in a correctly-initialized adapt object pointer created by thfAdaptCreate.
adapt object not initialized A non-NULL pointer was passed in for the adapt object, but this pointer does not contain a properly-initialized adapt object that is the output of thfAdaptCreate.
model not specified A NULL pointer was passed in for the trigger model. This pointer must be a valid pointer returned by thfAdaptModelGetTrigger.
search not valid The trigger model contains a pointer to an invalid search object.
recognizer not valid The trigger model contains a pointer to an invalid recognizer object.
failed to write search to file A file write error. Please make sure that the filename is valid and is available for writing, and that there is sufficient space available on the output device.
failed to write recognizer to file A file write error. Please make sure that the filename is valid and is available for writing, and that there is sufficient space available on the output device.
