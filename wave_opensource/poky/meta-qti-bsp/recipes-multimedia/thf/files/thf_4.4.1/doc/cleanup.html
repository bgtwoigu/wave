Cleaning Up

When you no longer need the recog, search or pronun objects, destroy them, even the ones created from static memory, with the following calls: thfRecogDestroy, thfSearchDestroy, and thfPronunDestroy.

Additionally, certain documented function return pointers to SDK-allocated memory that must be freed using thfFree.

Finally, don't forget to destroy the SDK session (thfSessionDestroy) when completely done with the SDK.
