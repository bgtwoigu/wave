thfSpeakerConfigGet

Syntax

#include <trulyhandsfree.h>
int thfSpeakerConfigGet(thf_t **thf, recog_t *r, unsigned short sID, 
                        thfSpeakerConfig_t key, float *value);

Description

Used to get a speaker verification or speaker ID parameter of the recognition object. A parameter is obtained by passing a key to indicate which parameter, and a pointer that will hold the parameter's current value. Only one parameter value can be obtained per call to this function. Values are returned as floating-point numbers.

Valid keys to use for parameters, and their meanings, are in the enumerated type thfSpeakerConfig_t.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
unsigned short sID in The identification (integer value) of the speaker whose parameters are to be read. For SPEAKER_TRIGGER_SAMP_PER_CAT, SPEAKER_SAMP_PER_CAT_WITHIN, SPEAKER_TARGET_SNR, SPEAKER_DUR_MIN_FACTOR, SPEAKER_DUR_MAX_FACTOR, and SPEAKER_DSP_SV_THRESH this value is ignored; the parameter value that is returned applies to all speakers.
thfSpeakerConfig_t key in Specification of which parameter value is to be returned.
float * value out The value of the specified parameter.

Return

Returns 1 on success or 0 on error.

Errors

String Description
recog object not found Invalid recognizer object handle.
speaker object not yet initialized Invalid speaker object handle.
unknown get option The value of getName does not indicate a valid parameter.
can't find subject ID Invalid speaker ID parameter
