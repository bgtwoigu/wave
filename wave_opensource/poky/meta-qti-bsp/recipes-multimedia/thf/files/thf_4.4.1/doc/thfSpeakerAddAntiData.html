thfSpeakerAddAntiData

Syntax

#include <trulyhandsfree.h>
int thfSpeakerAddAntiData(thf_t **thf, recog_t *r, unsigned short sID, 
                          short wantExamples, unsigned short useMonocat);

Description

Adds the specified number of training examples from the anti-speaker model into the internal list of recordings associated with the specified speaker ID.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
unsigned short sID in The identification (integer value) assigned to all anti-speaker data. This identification must have been already created in thfSpeakerInit or thfSpeakerAdd.
short wantExamples in The number of training examples to be selected from the anti-speaker model. The value must be greater than zero. Typical values are 512 or 1024, and usually this is the same value as 'trainSampWant' in thfSpeakerAdapt.
unsigned short useMonocat in Whether or not isolated phonetic categories are to be selected. If this value is '1', then categories are selected without regard to phonetic context. If this value is '0', then categories are selected taking phonetic context into account. Typically, when using the generic anti-speaker model, useMonocat is 0, and when using a phrase-specific anti-speaker model, useMonocat is 1.

Return

Returns 1 on success or 0 on error.

Errors

String Description
recog object not found Invalid recognizer object handle.
phrasespot search object not found Invalid phrasespot search object.
speaker object not yet initialized Invalid speaker object handle.
internal error: recog ROM struct not found Unexpected internal error. Contact customer support.
internal error: search ROM struct not found Unexpected internal error. Contact customer support.
internal error: search ROM names not found Unexpected internal error. Contact customer support.
