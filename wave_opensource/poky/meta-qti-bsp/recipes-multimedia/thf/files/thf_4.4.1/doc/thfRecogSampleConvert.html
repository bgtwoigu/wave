thfRecogSampleConvert

Syntax

#include <trulyhandsfree.h>
int thfRecogSampleConvert(thf_t **thf, recog_t *r, unsigned long inrate, 
                          short *inbuf, unsigned long inlen, short *outbuf, 
                          unsigned long *outlen);

Description

Converts the sample rate of the specified audio block to match the sample rate of the recognizer. Use thfFree to free "outbuf" memory when you're done.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
unsigned long inrate in Sample rate of input audio.
short * inbuf in Block of 16-bit linear PCM audio.
unsigned long inlen in Number of samples of input audio.
short * outbuf out Block of 16-bit linear PCM audio. Ownership of the associated memory is transferred to the application. It should be managed by the application and eventually deallocated using the thfFree function.
unsigned long * outlen out Number of samples in converted output audio.

Return

Returns 1 to indicate success and 0 for an error. Use thfGetLastError to get error message.

Errors

String Description
recog object not found Invalid recognizer object handle.

See Also

thfFree, thfRecogCreateFromFile, thfRecogCreateFromFunc and thfRecogCreateFromStatic
