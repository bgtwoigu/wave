thfSpeakerReadAntiSpeakerData

Syntax

#include <trulyhandsfree.h>
int thfSpeakerReadAntiSpeakerData(thf_t **thf, recog_t *r, char *filename);

Description

Read the pre-generated set of anti-speaker data from file. A phrase-specific set of anti-speaker data yields higher accuracy, but the generic anti-speaker data file can be used with any phrase. The recognizer object must already have been initialized for speaker verification or ID through a call to thfSpeakerInit. Anti-speaker data is generated from, and therefore tied to, a specific recognizer. Care should be taken to use the correct anti-speaker data file for a given recognizer; if there is a mis-match between the anti-speaker data file and the recognizer, an error will be returned.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
char * filename in Name of file that anti-speaker data are to be read from.

Return

Returns 1 on success or 0 on error.

Errors

String Description
recog object not found Invalid recognizer object handle.
filename passed to thfSpeakerReadAntiSpeakerData is NULL Invalid filename parameter
unable to open anti-speaker data file File open failed
internal error: recog ROM struct not found Internal error: Contact customer support
speaker object not yet initialized Invalid speaker object handle
no data in anti-data file No anti-speaker data found
no categories in anti-data file No anti-speaker categories found
mismatch between recognizer acoustic model and anti-data model Incompatible acoustic model and anti-data model
