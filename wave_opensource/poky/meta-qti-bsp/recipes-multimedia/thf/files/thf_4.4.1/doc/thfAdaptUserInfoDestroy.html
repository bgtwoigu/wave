thfAdaptUserInfoDestroy

Syntax

#include <trulyhandsfree.h>
int thfAdaptUserInfoDestroy(thf_t **thf, thfAdaptUserInfo_t *in);

Description

Destroy an object containing information about all users in a task. This object is of type thfAdaptUserInfo_t and it was created with a call to thfAdaptUserInfo.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
thfAdaptUserInfo_t * in userInfo user information object.

Return

Returns 1 on success or 0 on error.

Errors

String Description
userInfo to be destroyed is NULL A NULL pointer was passed in for the user information object. This must be a valid pointer to an object created with thfAdaptUserInfo.

thfAdaptCreate, thfAdaptTaskCreate, thfAdaptUserAdd, thfAdaptUserDestroy and thfAdaptUserInfo
