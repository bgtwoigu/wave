thfWaveSaveToFile

Syntax

#include <trulyhandsfree.h>
int thfWaveSaveToFile(thf_t **thf, const char *filename, short *speech, 
                      unsigned long len, unsigned long samplerate);

Description

Saves a block of 16-bit PCM audio samples to a RIFF wave file.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
const char * filename in Name of RIFF wave file to save.
short * speech in Block of 16-bit linear PCM audio samples.
unsigned long len in Number of samples in speech.
unsigned long samplerate in Sample rate of audio samples.

Return

Returns 1 to indicate success and 0 for an error. Use thfGetLastError to get error message.

Errors

String Description
Error writing output file The output file could not be written to. The file system denied access for writing to the datafile.
Could not open file The file name is not valid on this operating system, or the file is in use and no shared access is allowed on this operating system, or the application had insufficient privileges to open the file for writing.
