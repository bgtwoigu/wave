thfRecogCreateStub

Syntax

#include <trulyhandsfree.h>
recog_t *thfRecogCreateStub(thf_t **thf, unsigned long samplerate);

Description

Creates a recognizer object that can only be used for sample rate conversion.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
unsigned long samplerate in target sampling rate.

Return

Returns a unique handle to the recognizer object (see recog_t). Returns NULL in the event of error.

See Also

thfRecogSampleConvert and thfRecogDestroy
