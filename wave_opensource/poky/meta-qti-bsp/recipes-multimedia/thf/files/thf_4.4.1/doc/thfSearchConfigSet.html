thfSearchConfigSet

Syntax

#include <trulyhandsfree.h>
int thfSearchConfigSet(thf_t **thf, recog_t *r, searchs_t *s, 
                       thfSearchConfig_t key, float value);

Description

Used to set various search object parameters that change the default behavior. A parameter is set by passing a key to identify it, and a value that the parameter will be set to.

Valid keys are in the enumerated type thfSearchConfig_t.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object
recog_t * r in Recognizer object to which the search object relates.
searchs_t * s in Search object to be queried or set.
thfSearchConfig_t key in Identifies parameter to set
float value in New parameter value

Return

Returns 1 to indicate success and 0 for an error. Use thfGetLastError to get error message.

Errors

String Description
unknown thfSearchConfig_t key They key parameter, used to identify the search parameter to set, is not valid.
