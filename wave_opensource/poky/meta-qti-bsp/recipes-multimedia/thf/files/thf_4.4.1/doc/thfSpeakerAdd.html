thfSpeakerAdd

Syntax

#include <trulyhandsfree.h>
int thfSpeakerAdd(thf_t **thf, recog_t *r, unsigned short sID);

Description

Adds space in the recognizer object for another (unique) speaker (for speaker identification) or for speaker verification anti-speaker data (as described in How to Perform Speaker Verification.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
unsigned short sID in The identification (integer value) assigned to the new speaker

Return

Returns 1 on success or 0 on error.

Errors

String Description
recog object not found Invalid recognizer object handle.
speaker object not yet initialized Invalid speaker object handle.
internal error: failed to add speaker Unexpected internal error. Contact customer support.
