thfRecogDestroy

Syntax

#include <trulyhandsfree.h>
void thfRecogDestroy(recog_t *r);

Description

Destroys the specified recognizer object and frees associated memory. A search object currently associated with this recognition object through a previous call to thfRecogInit is not destroyed and should be destroyed separately.

Parameters

Type Name Dir. Description
recog_t * r in Recognizer object.

Return

None.

See Also

thfRecogCreateFromFile, thfRecogCreateFromFunc and thfRecogCreateFromStatic
