thfUdtConfigGet

Syntax

#include <trulyhandsfree.h>
int thfUdtConfigGet(thf_t **thf, udt_t *udt, thfUdtConfig_t key, float *value);

Description

Obtain one of the parameters used in UDT enrollment. Valid keys to use for parameters, and their meanings, are in the enumerated type thfUdtConfig_t.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
udt_t * udt in Udt object.
thfUdtConfig_t key in specification of which parameter to obtain
float * value out value of parameter

Return

Returns 1 on success or 0 on error.

Errors

String Description
udt object not found Invalid udt object handle.
unknown key The key parameter, used to identify the UDT parameter to retrieve, is not valid.

See Also

thfUdtCreate, thfUdtConfigSet and constants
