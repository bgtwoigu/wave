thfUdtCreate

Syntax

#include <trulyhandsfree.h>
udt_t *thfUdtCreate(thf_t **thf, char *recogFile, char *phnSearchFile, 
                    char *udtDataFile, unsigned short maxNumEnroll, 
                    unsigned long samplerate);

Description

Create a user-defined trigger (UDT) object from recognizer, phoneme-level search, and data files.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
char * recogFile in file containing general-purpose recognizer.
char * phnSearchFile in file containing phoneme search object.
char * udtDataFile in file containing data for UDT enrollment.
unsigned short maxNumEnroll in maximum number of enrollment recordings per user
unsigned long samplerate sampling frequency used in recognition and for enrollment recordings

Return

Returns a pointer to a udt_t object on success, or NULL on error.

Errors

String Description
Need at least one enrollment per user, not zero The value of numEnroll is zero. There must be at least one enrollment recording per user.
Failed to create phoneme recognition object from file The call to thfUdtCreate did not specify a file that contains a valid recognizer.
Failed to create phoneme search object from file The call to thfUdtCreate did not specify a file that contains a valid phoneme search object.
Failed to initialize phoneme recognition object An internal call to thfPhonemeRecogInit failed; contact customer support.
Failed to create trigger recognition object from file Internal error: contact customer support
Trigger recognizer is incompatible with audio samplerate The recognizer uses a different sampling frequency than the value of 'samplerate'.
Error reading UDT data file The call to thfUdtCreate did not specify a file that contains a valid UDT+SV/SID or SV/SID data object.
