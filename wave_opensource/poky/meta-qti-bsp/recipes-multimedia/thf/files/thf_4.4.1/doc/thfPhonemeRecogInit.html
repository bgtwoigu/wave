thfPhonemeRecogInit

Syntax

#include <trulyhandsfree.h>
int thfPhonemeRecogInit(thf_t **thf, recog_t *r, phsearch_t *ps, char keep);

Description

Initializes a phoneme recognizer object and associates it with a phoneme search object. The information that gets returned by thfRecogResult is configured using the keep parameter. The results always include the recognized word/s and associated score. The following flags are used to request additional information:
RECOG_KEEP_NONEno additional information (i.e., only word/s and score)
RECOG_KEEP_WAVEreturns input wave and sdet wave
RECOG_KEEP_WAVE_WORDreturns input wave, sdet wave, and word alignments
RECOG_KEEP_WAVE_WORD_PHONEMEreturns input wave, sdet wave, word alignments and phoneme alignments
RECOG_KEEP_WORD_PHONEMEreturns word alignments and phoneme alignments
For phoneme recognition, the typical value of 'keep' is RECOG_KEEP_WAVE_WORD_PHONEME.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
phsearch_t * ps in Phoneme search object.
char keep in parameter indicating what kind of data to return from thfRecogResult. The typical value is RECOG_KEEP_WAVE_WORD_PHONEME.

Return

Returns 1 on success or 0 on error.

Errors

String Description
no recog object Invalid recognizer object handle.
no phonemerec object Invalid phoneme search object handle
