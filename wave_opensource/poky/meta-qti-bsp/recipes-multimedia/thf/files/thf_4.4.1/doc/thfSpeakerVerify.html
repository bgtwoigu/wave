thfSpeakerVerify

Syntax

#include <trulyhandsfree.h>
int thfSpeakerVerify(thf_t **thf, recog_t *r, unsigned short sID, 
                     float *spkrRelativeScore, float *spkrRawScore, 
                     short *recordingOK);

Description

Perform verification of the last trigger identified by thfRecogResult against the model associated with the specified speaker ID. Larger values indicate greater confidence that the utterance was spoken by the specified speaker.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
unsigned short sID in The identification (integer value) of the speaker whose model is to be compared against the trigger result.
float * spkrRelativeScore out The relative score of how well the result from thfRecogResult matches the specified speaker. Larger values indicate a better match. These values usually range from -1.0 to 1.0. In general, spkrRelativeScore is considered not as good as spkrRawScore (below), although the correlation between the two is high.
float * spkrRawScore out Another speaker verification score of how well the result from thfRecogResult matches the specified speaker. Larger values indicate a better match. These values range from 0.0 to 1.0. In general, spkrRawScore is considered a better measure than spkrRelativeScore, although the correlation between the two is high.
short * recordingOK in/out Whether or not to perform a check for a sufficient signal-to-noise ratio (SNR) in the recording. If 'recordingOK' is NULL, then this check is not performed, and verification is always performed. If 'recordingOK' is a valid pointer to a 'short' variable, then this check is performed. If the SNR is above a fixed, pre-determined threshold, then the value of 'recordingOK' is set to 1 and 'spkrRelativeScore' contains a valid result. If the SNR is below this threshold, then the value of 'recordingOK' is set to 0 and 'spkrRelativeScore' is set to 0.0.

Return

Returns 1 on success or 0 on error.

Errors

String Description
recog object not found Invalid recognizer object handle.
phrasespot search object not found Invalid phrasespot search object.
speaker object not yet initialized Invalid speaker object handle.
internal error: recog ROM struct not found Unexpected internal error. Contact customer support.
internal error: search ROM struct not found Unexpected internal error. Contact customer support.
internal error: recog ROM struct acoustic model not found Unexpected internal error. Contact customer support.
internal error: failed to verify Unexpected internal error. Contact customer support.
