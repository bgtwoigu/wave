pseval

Syntax

TrulyHandsfree SDK\<platform>\bin\pseval [options] -n netfile -s searchfile wavefile

Description

An evaluation tool for a phrasespot recognizer. Given an acoustic model, search, and an audio file, print to stdout a time-stamped list of all occurrences of spotted phrases. Optionally displays word and phoneme alignments, and speaker verification scores.

When the input to pseval comes from the output of enroll, the console output from enroll indicates recommended values for several of the optional pseval parameters.

Parameters

Name Default Description
-n netfile Neural network (acoustic model) file name.
-s searchfile Compiled phrase-spotting search file name.
wavefile Audio file. This can be in RIFF (.wav) format with 16-bit linear PCM encoding, or a headerless 16-bit linear PCM file in host byte order. If the latter, the -r rate flag is required to specify the sample rate.
-a offset 0 Sets the phrase spotting param A offset
-d delay 300 Sets the phrase spotting delay in ms.
-m thr -5.0 Sets the EPQ minimum threshold.
-e 0|1 0 Enable (1) or disable (0) EPQ; see epqEnable. The default is 0 (not enabled).
-f disabled Disable audio check and force evaluation.
-r rate read from wavfile Audio sample rate in Hz, required if the wave file is in PCM format.
-t threshold disabled Sets the speaker verification (SV) score threshold. SV scores will only be printed if this option is given. If the search does not support SV, an SV score of -1000 will be returned.
-p disabled Show phoneme alignments.
-q disabled Show EPQ SNR.
-w disabled Show word alignments.
-v disabled Produce verbose progress messages.
-x disabled Resets the recognizer after each spot. This will also reset the alignment time stamps.
