thfUdtCheckValueGet

Syntax

#include <trulyhandsfree.h>
int thfUdtCheckValueGet(thf_t **thf, udt_t *udt, thfUdtConfig_t key, 
                        float *val, unsigned short numEnroll);

Description

Get array of enrollment quality feedback values calculated during thfUdtCheckEnrollments. Valid keys to use for parameters are in the enumerated type thfUdtConfig_t with the "UDT_CHECK_" prefix.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
udt_t * udt in UDT object.
thfUdtConfig_t key in Specification of which parameter to get the values of.
float * val out Array of floats to contain the feedback value for each enrollment. Array must have an element per enrollment file allocated.
unsigned short numEnroll in Number of enrollment files.

Return

Returns 1 on success or 0 on error.

Errors

String Description
udt object not found Invalid udt object handle.
feedback check values not yet calculated, call thfSpeakerCheckEnrollments Function called prior to thfSpeakerCheckEnrollments.
unknown option The 'key' parameter, used to identify the UDT_CHECK_ parameter to retrieve, is not valid.
