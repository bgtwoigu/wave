Constants

The following enumerations and constants are defined in the public header file and are used to modify SDK behavior through configuration functions.

SDK version

These TrulyHandsfree SDK version information is available:
Identifier Description
THF_VERSION_MAJOR Major version number, incremented when large, backwards-incompatible API changes are made.
THF_VERSION_MINOR Minor version number, incremented when new features are added to the API.
THF_VERSION_PATCH Patch number, incremented for bug-fix releases that do not alter the API.
THF_VERSION String containing "THF_VERSION_MAJOR.THF_VERSION_MINOR.THF_VERSION_PATCH", e.g.: "4.4.1"

Text-To-Speech Configuration

thfTtsConfig_t key values, used with thfTtsConfigGet and thfTtsConfigSet.
IdentifierDefaultDescription
PROSODY_DOMAIN GENERIC The type of prosody to use, either GENERIC for general-purpose prosody, or NAMES for caller-ID prosody.
PROSODY_PITCHRANGE FULL FULL to use the default pitch range, or HALF to reduce pitch excursions by 50%.
