Sensory Confidential

combine

Syntax

TrulyHandsfree SDK\<platform>\bin\combine [options] -n netfile1 -N netfile2 -s searchfile1 -S searchfile2 -x outputnetfile -y outputsearchfile

Description

Combine two recognizers (acoustic models and phrase spot searches) into a single recognizer, with one acoustic model and one search.

The -p and -P options are used to supply string prefixes added to the results from the first and second searches, respectively. These can be used to disambiguate results from the two searches, should these produce results that use the same tag. (E.g. when combining a speaker-independent "Hello Blue Genie" trigger with a enrolled speaker verification version of the same phrase.)

Parameters

Name Default Description
-n netfile1 First neural network (acoustic model) file name.
-N netfile2 First neural network (acoustic model) file name.
-s searchfile1 First compiled phrase-spotting search file name.
-S searchfile2 Second compiled phrase-spotting search file name.
-x outputnetfile Output file name used for combined acoustic model.
-y outputsearchfile Output file name used for combined search.
-m mapfile Output file name used for phrase IDs in the combined search.
-p prefix1 Add the prefix1 string to searchfile1 results.
-P prefix2 Add the prefix2 string to searchfile2 results.
-v Show progress messages. If this flag is used and no phrase map file is specified with -m, the phrase map will be printed to stdout.
