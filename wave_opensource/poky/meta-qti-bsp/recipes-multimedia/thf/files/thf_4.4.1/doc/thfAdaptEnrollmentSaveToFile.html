thfAdaptEnrollmentSaveToFile

Syntax

#include <trulyhandsfree.h>
int thfAdaptEnrollmentSaveToFile(thf_t **thf, adaptObj_t *aObj, char *taskName, 
                                 char *userName, long enrollmentID, 
                                 char *fileName);

Description

Save the specified enrollment data (or all enrollments) to file.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
adaptObj_t * aObj in adapt object.
char * taskName in name of current task.
char * userName in name of user for whom the enrollments are being added.
long enrollmentID in unique identifier for the enrollment being saved to file. This ID is set by thfAdaptEnrollmentAddTrigger. If enrollmentID is ADAPT_UNSPECIFIED_ID (as defined in trulyhandsfree.h) then all enrollments will be saved.
char * fileName in name of file to which enrollment data will be written.

Return

Returns 1 on success or 0 on error.

Errors

String Description
adapt object not found A NULL pointer was passed in for the adapt object; pass in a correctly-initialized adapt object pointer created by thfAdaptCreate.
adapt object not initialized A non-NULL pointer was passed in for the adapt object, but this pointer does not contain a properly-initialized adapt object that is the output of thfAdaptCreate.
task name not specified A valid task name must be provided; a task with this name must have been created with a previous call to thfAdaptTaskCreate.
user name not specified A NULL pointer was passed in for the user name. A valid user name (a string of one or more ASCII characters, no spaces) must be provided.
file name not specified A NULL pointer was passed in for the output filename. A valid filename must be specified.
internal error creating keyed object An internal error. Please contact customer support.
failed to write enrollments data to file A file write error. Please make sure that the filename is valid and is available for writing, and that there is sufficient space available on the output device.
