thfAdaptEnrollmentCheckDestroy

Syntax

#include <trulyhandsfree.h>
int thfAdaptEnrollmentCheckDestroy(thf_t **thf, thfEnrollmentCheck_t *in);

Description

Destroy an object containing information about enrollment quality checks. This object is of type thfEnrollmentCheck_t and it was created with a call to thfAdaptEnrollmentCheck or thfAdaptEnrollmentQuickCheck.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
thfEnrollmentCheck_t * in result enrollment check object.

Return

Returns 1 on success or 0 on error.

See Also

thfAdaptCreate, thfAdaptTaskCreate, thfAdaptUserAdd, thfAdaptEnrollmentAddTrigger, thfAdaptEnrollmentCheck and thfAdaptEnrollmentQuickCheck
