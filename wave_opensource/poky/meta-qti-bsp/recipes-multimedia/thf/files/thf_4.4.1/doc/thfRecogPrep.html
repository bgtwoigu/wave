thfRecogPrep

Syntax

#include <trulyhandsfree.h>
int thfRecogPrep(thf_t **thf, recog_t *r, unsigned long inlen, short *inbuf);

Description

Feeds supplied audio through the feature extractor to stabilize the background mean estimation. This is particularly useful for batch testing, where audio files may have been captured in different environments. It forces the feature extractor to adapt to the new environment thereby better approximating live audio conditions. Typically only used before calling thfRecogPipe when recognizing audio from file.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
unsigned long inlen in Number of samples of input audio.
short * inbuf in Block of 16-bit linear PCM audio.

Return

Returns 1 on success or 0 on error.

Errors

String Description
recog object not found Invalid recognizer object handle.
