thfSpeakerRemove

Syntax

#include <trulyhandsfree.h>
int thfSpeakerRemove(thf_t **thf, recog_t *r, unsigned short sID);

Description

Remove the data, model, and all information associated with a speaker from the recognizer object. The speaker ID that was used is then free to be added again with a call to thfSpeakerAdd.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
unsigned short sID in The identification (integer value) of the speaker to be removed from the recognizer object.

Return

Returns 1 on success or 0 on error.

Errors

String Description
recog object not found Invalid recognizer object handle.
speaker object not yet initialized Invalid speaker object handle
internal error: failed to remove speaker Internal error: Contact customer support
