TrulyHandsfree Software Development Kit

Version 4.4

www.sensory.com

Sensory's TrulyHandsfree engine is a small-footprint, speaker-independent speech recognizer. This SDK enables developers like you to easily integrate speech recognition into their applications. It provides C-based API definitions, code libraries, tools, and samples for using the TrulyHandsfree engine.

Sensory's TrulyHandsfree engine is ideal for consumer electronics, mobile and embedded devices, automotive, telephony devices, and PC-based applications.

The SDK now includes the latest TrulyHandsfree™ Voice Control 4.0 technology.
