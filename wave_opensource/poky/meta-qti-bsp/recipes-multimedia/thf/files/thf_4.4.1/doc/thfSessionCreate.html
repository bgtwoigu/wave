thfSessionCreate

Syntax

#include <trulyhandsfree.h>
thf_t *thfSessionCreate(void);

Description

Creates an SDK session object that encapsulates session-specific data. You must create a session object before using other SDK API calls and destroy the session object when you've finished with the SDK (see thfSessionDestroy).

Parameters

None.

Return

A pointer to the session object (see thf_t), or NULL on error.
