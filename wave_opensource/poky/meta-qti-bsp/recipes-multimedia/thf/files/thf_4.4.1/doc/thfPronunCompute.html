thfPronunCompute

Syntax

#include <trulyhandsfree.h>
const char *thfPronunCompute(thf_t **thf, pronuns_t *p, const char *word, 
                             unsigned short n, const char *suffix, 
                             const char *pos);

Description

Computes a pronunciation string for the specified word or phrase. The number of pronunciations returned for each word/phrase can be specified however a single pronunciation is usually sufficient. If the phrase contains multiple words, this function does not insert silence (or optional silence) between words in the resulting pronunciation. The supplied suffix is appended to the pronunciation string. This is typically either a 'required' or 'optional' . .pau . and/or . .any ., or NULL if no suffix is required. Part-of-speech tags can be used to differentiate between alternate syntactic forms of a word. For instance, the pronunciation for the word 'record' in English may differ when used as a 'noun' versus a 'verb'. Part-of-speech values depend on the language and listed in the language-specific SDK documentation. The following values are supported where applicable:
ValueDescription
adjADJECTIVE
advADVERB
prePREPOSITION
proPRONOUN
verbVERB
nounNOUN
interjectionINTERJECTION
determinentDETERMINENT
conjunctionCONJUNCTION

NOTE: The returned string represents a 'base' pronunciation, including syllable boundaries and stress markings. Phonology rules are applied to this base pronunciation when creating the search object using thfSearchCreateFromList or thfSearchCreateFromGrammar. The phonology rules account for common pronunciation variations that result from contextual effects and regional dialects.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
pronuns_t * p in Pronunciation object.
const char * word in Word or phrase for which to compute a pronunciation. Encoding is UTF-8.
unsigned short n in Target number of unique pronunciations to be computed.
const char * suffix in Suffix string to be added to end of pronunciation string, or NULL if no suffix is needed.
const char * pos in Part-of-speech of word or NULL.

Return

Returns a pronunciation string for input word or phrase. The string belongs to, and its memory is managed by, the SDK. Returns NULL in the event of an error.

Errors

String Description
failed for word Supplied word could not be decoded (assuming UTF-8) or contained a character that does not exist in the language writing system.
