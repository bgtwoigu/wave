thfSearchCreateFromFile

Syntax

#include <trulyhandsfree.h>
searchs_t *thfSearchCreateFromFile(thf_t **thf, recog_t *r, 
                                   const char *searchfile, 
                                   unsigned short nbest);

Description

Reads pre-compiled search from file and creates a search object. The pre-compiled search and the recognition object must have the same frame size, otherwise an error is returned. Each object has a unique handle. Call thfSearchDestroy to destroy the search object when no longer needed.

NOTE: Pre-built compiled searches provided by Sensory may contain additional information that cannot be expressed in textual form with thfSearchCreateFromGrammar, and may result in different recognition behavior.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognition object.
const char * searchfile in File containing pre-compiled search.
unsigned short nbest in Number of recognition results required

Return

Returns a unique handle to the search object (see searchs_t). Returns NULL in the event of error.

Errors

String Description
recog object not found Invalid recognizer object handle.
Could not open raw file The datafile does not exist or the application had insufficient privileges to open the file for reading.
Error reading raw file: The datafile did not decode correctly. Was the file damaged?
net and search have different frame rates Incompatible acoustic model and search objects have been used in combination during initialization.
Unknown search type in file Data file could not be decoded. Probably an older or newer version.
