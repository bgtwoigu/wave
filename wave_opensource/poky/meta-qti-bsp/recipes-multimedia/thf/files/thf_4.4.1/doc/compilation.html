Developing an Application Using the SDK

Sensory's TrulyHandsfree SDK is written entirely in C. It can be readily integrated into applications written in C, C++, C#, Tcl/Tk and Visual Basic, amongst other languages.

It is assumed that you have a suitable compiler and, if available, integrated development environment (IDE) correctly installed, such as Microsoft Visual Studio, and that you are familiar with how to program using the IDE or underlying compiler. Most IDEs ship with sample code which you should be comfortable with before attempting to use this SDK.

Next, make sure you familiarize yourself with the set of examples provided as part of this SDK (see Sample Code).

Now you're ready to begin developing with applying the SDK. To integrate SDK API functions into your C/C++ application:

  1. Include the trulyhandsfree.h SDK header file in your application.
  2. Use SDK API functions in your application.
  3. Link to the appropriate SDK static or dynamic code library.
NOTE: Libraries are platform specific and contain different subsets of API functions, so be sure to link with the appropriate one.
