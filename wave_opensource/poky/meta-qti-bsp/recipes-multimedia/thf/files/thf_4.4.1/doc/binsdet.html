sdet

Syntax

TrulyHandsfree SDK\<platform>\bin\sdet [options] -n netfile -i inputwavefile -o outputwavefile

Description

Takes a net file and input wave file, writes the speech-detected portion of the input wave file to an output wave file.

Parameters

Name Default Description
-n netfile Neural network (acoustic model) file name.
-i inputwavefile Wave file to perform speech detection on. This can be in RIFF (.wav) format with 16-bit linear PCM encoding, or a headerless 16-bit linear PCM file in host byte order. If the latter, the -r rate flag is required to specify the sample rate.
-o outputwavefile Output wave file containing detected speech of input file.
-d mind 100.0 Sets the minimum duration parameter. parameter.
-m mr 10000.0 Sets the maximum record time parameter.
-l ls 5000.0 Sets the pre-speech silence parameter.
-t ts 400.0 Sets the trailing silence parameter.
-L ltw 1500.0 Sets the long term window size parameter.
-S stw 60.0 Sets the short term window size parameter.
-E espt 0 Sets the speech energy threshold parameter.
-e espt 0.4 Sets the silence energy threshold parameter.
-r rate read from wavfile Audio sample rate in Hz, required if the wave file is in PCM format.
-v disabled Produce verbose progress messages.
