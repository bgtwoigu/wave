thfSpeakerCreateFromFile

Syntax

#include <trulyhandsfree.h>
int thfSpeakerCreateFromFile(thf_t **thf, recog_t *r, unsigned short numEnroll, 
                             char *filename);

Description

Initializes structures and adds data for speaker verification (SV), speaker identification (SID), and/or user-defined triggers (UDT) to a recognizer object.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
unsigned short numEnroll in The desired number of recordings for speaker verification
char * filename in data file for enrolling SV/SID, UDT, or both.

Return

Returns 1 on success or 0 on error.

Errors

String Description
recog object not found Invalid recognizer object handle.
recognizer is already associated with a speaker object A speaker object has already been associated with this recognizer. A recognizer can only be associated with a single speaker object.
internal error: recog ROM struct not found Internal error: Contact customer support
filename passed to thfSpeakerCreateFromFile is NULL A valid filename of a SV/SID and/or UDT data file must be specified.
unable to open data file A valid SV/SID and/or UDT data file must be specified.
thfRecogInit keepwave arg must be RECOG_KEEP_WAVE_WORD_PHONEME When the specified recognizer was initialized, it must be initialized with the RECOG_KEEP_WAVE_WORD_PHONEME flag.
internal error: error initializing speaker object Internal error: Contact customer support
mismatch between recognizer acoustic model and anti-data model The data in the specified SV/SID file does not match the acoustic model in the specified recognizer.
