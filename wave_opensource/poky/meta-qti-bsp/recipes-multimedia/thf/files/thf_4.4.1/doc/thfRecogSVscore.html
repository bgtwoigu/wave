thfRecogSVscore

Syntax

#include <trulyhandsfree.h>
float thfRecogSVscore(thf_t **thf, recog_t *r);

Description

Returns a speaker-verification score that is computed during a user-defined trigger (UDT) phrase spotting search. This function can be called immediately after thfRecogResult. If the search is not a UDT search, a value of -1000.0 is returned. This score can be compared against a pre-determined threshold to decide if the detected word was spoken by the enrolled user.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.

Return

Returns speaker verification score.
