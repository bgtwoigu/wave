thfAdaptEnrollmentAddTrigger

Syntax

#include <trulyhandsfree.h>
int thfAdaptEnrollmentAddTrigger(thf_t **thf, adaptObj_t *aObj, char *taskName, 
                                 char *userName, 
                                 thfTriggerEnrollment_t *enroll);

Description

Add the enrollment information (specified in an enrollment object, described by the structure in thfTriggerEnrollment_t) to the specified user in the specified task. This function returns, in the enrollment object, an ID that uniquely identifies this enrollment within the task. The enrollment object must be created, populated, and destroyed by the application.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
adaptObj_t * aObj in adapt object.
char * taskName in name of current task.
char * userName in name of user for whom the enrollments are being added.
thfTriggerEnrollment_t * enroll in object containing enrollment information.

Return

Returns 1 on success or 0 on error.

Errors

String Description
adapt object not found A NULL pointer was passed in for the adapt object; pass in a correctly-initialized adapt object pointer created by thfAdaptCreate.
adapt object not initialized A non-NULL pointer was passed in for the adapt object, but this pointer does not contain a properly-initialized adapt object that is the output of thfAdaptCreate.
task name not specified A valid task name must be provided; a task with this name must have been created with a previous call to thfAdaptTaskCreate.
user name not specified A NULL pointer was passed in for the user name. A valid user name (a string of one or more ASCII characters, no spaces) must be provided.
enrollment data not specified A NULL pointer was passed in for the enrollment data object. A valid enrollment object must be provided.
invalid sampling frequency The sampling frequency specified in the enrollment object is not valid.
enrollment audio has no data The audio data specified in the enrollment object is either a NULL pointer or has zero length.
reserved data not properly set The variables in the enrollment object called 'reserved' must be set to NULL.
input waveform has sampling frequency XX Hz; task requires YY Hz The enrollment has a sampling frequency that is lower than the sampling frequency required by the task. Change the sampling frequency used to record the enrollment audio.
