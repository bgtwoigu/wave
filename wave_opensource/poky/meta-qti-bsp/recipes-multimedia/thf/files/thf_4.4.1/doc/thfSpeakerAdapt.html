thfSpeakerAdapt

Syntax

#include <trulyhandsfree.h>
int thfSpeakerAdapt(thf_t **thf, recog_t *r, unsigned short sID, 
                    unsigned short trainMethod, char *ignorePhnSpec, 
                    unsigned short trainSampWant);

Description

Adapt, or train, a model to learn the characteristics of a speaker's voice. This function does NOT check whether the number of recordings enrolled through thfSpeakerStoreLastRecording is equal to the desired number of recordings specified in thfSpeakerInit.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
unsigned short sID in The identification (integer value) of the speaker data to be learned
unsigned short trainMethod in The only currently-supported method is for a value of 0, which is appropriate for speaker verification and speaker identification.
char * ignorePhnSpec in A string that specifies what types of phonemes to ignore during adaptation. Supported values are 'IGNORE_SV_DEFAULT' and 'IGNORE_PAUSE'. 'IGNORE_SV_DEFAULT' will cause the function to ignore phonemes in the recording that are less important for speaker verification and speaker ID, e.g. the /f/ sound in 'foo'. 'IGNORE_PAUSE' will cause the function to ignore only the 'pause' phoneme. Performance is typically better with 'IGNORE_SV_DEFAULT', but 'IGNORE_PAUSE' is made available for experimentation.
unsigned short trainSampWant in The number of training samples to be used in adaptation. Typical values are 512 or 1024. Smaller values reduce the time required to learn the speaker's voice characteristics, usually at the expense of verification accuracy. Larger values increase the time required for training and may (or may not) improve verification performance. A value of 0 is not supported.

Return

Returns 1 on success or 0 on error.

Errors

String Description
recog object not found Invalid recognizer object handle.
internal error: recog ROM struct not found Unexpected internal error. Contact customer support.
internal error: search ROM names not found Unexpected internal error. Contact customer support.
internal error: recog ROM struct acoustic model not found Unexpected internal error. Contact customer support.
speaker object not yet initialized Invalid speaker object handle.
internal error: failed to adapt Invalid speaker object handle.
