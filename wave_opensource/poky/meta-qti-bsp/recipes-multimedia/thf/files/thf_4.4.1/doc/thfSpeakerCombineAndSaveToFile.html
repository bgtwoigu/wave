thfSpeakerCombineAndSaveToFile

Syntax

#include <trulyhandsfree.h>
int thfSpeakerCombineAndSaveToFile(thf_t **thf, recog_t *r, searchs_t *s1, 
                                   searchs_t *s2, unsigned short sID, 
                                   char *searchFile, char *recogFile);

Description

Processes a user-specific search and recognizer for application as a user-defined trigger (UDT), or combines two user-specific searches and recognizers for application as a multi-user UDT trigger. The resulting search and recognizer are saved to the specified files.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
searchs_t * s1 in either NULL (for a single or initial UDT) or Search object created by thfUdtEnroll
searchs_t * s2 in Search object created by thfUdtEnroll
unsigned short sID in The identification (integer value) of the speaker associated with s2.
char * searchFile out the filename where the UDT search object will be written
char * recogFile out the filename where the UDT recognizer object will be written

Return

Returns 1 on success or 0 on error.

Errors

String Description
recog object not found Invalid recognizer object handle.
empty search objects Both s1 and s2 are NULL; at least s2 must be a pointer to a search returned by thfUdtEnroll
second search cannot be NULL While the first search (s1) can be NULL, the second search (s2) must be a pointer to a search returned by thfUdtEnroll
internal error: no search ROM object in first search Internal error: Contact customer support
internal error: no spotting search object in second search Internal error: Contact customer support
internal error: no search ROM object in second search Internal error: Contact customer support
internal error: no search names in second search Internal error: Contact customer support
internal error: error in combining searches and models Internal error: Contact customer support
failed to write search to file The UDT search object could not be written to the specified filename.
failed to write recognizer to file The UDT recognizer object could not be written to the specified filename.
