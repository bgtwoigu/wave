thfWaveFromFile

Syntax

#include <trulyhandsfree.h>
int thfWaveFromFile(thf_t **thf, const char *filename, short **speech, 
                    unsigned long *len, unsigned long *samplerate);

Description

Reads a RIFF wave file into memory, reporting its sample rate and length in samples. The returned wave object must be freed with thfFree when no longer needed.
For stereo wave files, only the left channel is used.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
const char * filename in Name of file to load.
short ** speech out Block of 16-bit linear PCM audio. Ownership of the associated memory is transferred to the application. It should be managed by the application and eventually deallocated using the thfFree function.
unsigned long * len out Number of speech samples in speech.
unsigned long * samplerate out Sample rate of speech in speech.

Return

Returns 1 to indicate success and 0 for an error. Use thfGetLastError to get error message.

Errors

String Description
Could not open file The datafile does not exist or the application had insufficient privileges to open the file for reading.
not a RIFF waveform audio file The input audio file is not the correct file format.
waveform audio file has no fmt chunk. Malformed RIFF file.
fmt chunk read failed. Malformed RIFF file.
waveform audio file has no data chunk. Malformed RIFF file.
truncated data chunk Malformed RIFF file.
WAVE format (id <0x????>) not supported Malformed RIFF file.
