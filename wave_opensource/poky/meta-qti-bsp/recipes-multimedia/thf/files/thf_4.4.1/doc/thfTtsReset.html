thfTtsReset

Syntax

#include <trulyhandsfree.h>
void thfTtsReset(tts_t *tts);

Description

Reset Tts engine following partial or complete synthesis. Reset is required before resynthesizing additional phrases.

Parameters

Type Name Dir. Description
tts_t * tts in Tts object.

Return

nothing.
