thfPhrasespotCreateFromList

Syntax

#include <trulyhandsfree.h>
searchs_t *thfPhrasespotCreateFromList(thf_t **thf, recog_t *r, pronuns_t *p, 
                                       const char **phrases, 
                                       unsigned short phraseCount, 
                                       const char **words, 
                                       const char **pronuns, 
                                       unsigned short wordCount, 
                                       unsigned short listenWindow);

Description

Creates a phrasespot search object for the specified array of phrases. Phrasespot parameters (paramA, paramB) are automatically assigned based on the specified listening window duration. To further affect phrasespot accept/reject behavior, adjustments can be made to the PHRASESPOT_PARAMA_OFFSET parameter using thfPhrasespotConfigSet. The array of unique words contained in the phrases and their associated pronunciations can be optionally specified. If 'words' and 'pronun' arguments are set to NULL, pronunciations are automatically generated using the specified pronunciation model. The search object can be used immediately or saved to file for later use. Call thfSearchDestroy to destroy the search object when no longer needed.

The listening window duration directly affects phrasespotting behavior, such that a shorter listening duration will have a higher per-hour false accept rate than a longer listening duration. The listening window can have one of the following values:

  • PHRASESPOT_LISTEN_SHORT
  • PHRASESPOT_LISTEN_MEDIUM
  • PHRASESPOT_LISTEN_LONG
  • PHRASESPOT_LISTEN_CONTINUOUS

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
pronuns_t * p in Pronunciation object.
const char ** phrases in Array of phrases to build.
unsigned short phraseCount in Number of phrases in array.
const char ** words in Array of unique words in phrases. Can be NULL if pronunciations are to be generated internally to SDK.
const char ** pronuns in Array of word pronunciations. Can be NULL if pronunciations are to be generated internally to SDK.
unsigned short wordCount in Number of words and pronunciations in array (if provided).
unsigned short listenWindow in Duration of intended listening window (in seconds).

Return

Returns a unique handle to the search object (see searchs_t). Returns NULL in the event of error.

Errors

String Description
recog object not found Invalid recognizer object handle
pronun object not found Invalid pronunciation object handle.
Undefined acoustic model state names The acoustic model data seems to be damaged or incompatible.
Uninitialized pronunciation engine Invalid Pronunciation object handle.
must provide phrases. phraseCount cannot be zero Empty phrase list or phraseCount is zero.
word/pronun mismatch Word and pronun list must be of equal length.
