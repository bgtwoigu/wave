CreatePronun

Syntax

TrulyHandsfree SDK\<platform>\bin\createPronun ltsDataFile
   inputTextFile outputSrcFile variableName dialectIndex [nBest [Suffix]]

Description

Generates an ordered list of pronunciations from given vocabulary and stores them in C-language arrays, suitable for compilation into an application.

Parameters

Name Dir. Description
ltsDataFile in The language specific pronunciation model data file.
inputTextFile in The text file containing the Carriage-Return terminated words or phrases.
outputSrcFile out The output Source file to be created containing the vocabulary and pronunciation C-language arrays.
variableName in The variable name to be used for the resulting vocabulary and pronunciation arrays in the source file.
dialectIndex in Dialect Index. See corresponding language pack for supported dialects.
nBest in Optional target number of unique pronunciations to be created per phrase or word. The default nBest setting is 2.
Suffix in Optional suffix string to be added to end of the pronunciation string. The default suffix setting is an empty string.

See Also

thfSearchCreateFromList, thfSearchCreateFromGrammar.
