Full Text-to-Speech Synthesis

Full synthesis allows you to generate audio using the following simple steps :-

  1. Load a voice database using thfTtsCreateFromFile or thfTtsCreateFromStatic.
  2. Synthesize audio for input text using thfTtsSynthesize. This returns a single block of 16kHz 16-bit linear PCM audio and size (in samples). Input parameters are used to configure the pitch and speech rate.
  3. Call thfFree to free returned block of audio.
  4. Call thfTtsReset before re-synthesizing a new phrase.
  5. Call thfTtsDestroy when done synthesizing to clean up memory.
