thfUdtEnrollEstimateParams

Syntax

#include <trulyhandsfree.h>
int thfUdtEnrollEstimateParams(thf_t **thf, udt_t *udt, thfUser_t *user, 
                               unsigned short numUsers, 
                               unsigned short whichUser, float learnPS, 
                               float learnSV, float *paramAoffset, 
                               float *svThreshold);

Description

Do not use this experimental function without explicit recommendation from Sensory.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
udt_t * udt in Udt object.
thfUser_t * user in array of user data structures, containing all enrollment recordings for all users. See the data structure documentation.
unsigned short numUsers in number of users to enroll.
unsigned short whichUser in set to 0.
float learnPS in paramAoffset multiplier.
float learnSV in svThreshold multiplier.
float * paramAoffset out estimated paramAoffset.
float * svThreshold out estimated speaker verification threshold.

Return

Returns 1 on success or 0 on error.
