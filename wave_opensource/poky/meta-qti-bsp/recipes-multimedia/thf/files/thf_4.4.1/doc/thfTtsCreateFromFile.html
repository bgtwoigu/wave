thfTtsCreateFromFile

Syntax

#include <trulyhandsfree.h>
tts_t *thfTtsCreateFromFile(thf_t **thf, const char *voicefile);

Description

Reads in the voice file and creates a tts object.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
const char * voicefile in Voice filename.

Return

Returns a unique handle to the tts object (see tts_t). Returns NULL in the event of an error.

Errors

String Description
Unlicensed tts object Data file not properly licensed. Contact customer support.
Unexpected regexpC_t fields Internal error. Contact customer support.
Unexpected container: Internal error. Contact customer support.
