thfRecogGetClipping

Syntax

#include <trulyhandsfree.h>
int thfRecogGetClipping(thf_t **thf, recog_t *r, float *clipping);

Description

Reports the percentage of clipped speech seen by the internal speech detector. Clipping is when the audio amplitude exceeds the maximum input of the audio digitizer (corresponding to a digitized value of 32768), which distorts the audio spectrum. Clipped speech reduces recognition accuracy significantly.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
float * clipping out Percentage of speech that was clipped.

Return

Returns 1 to indicate success and 0 for an error. Use thfGetLastError to get error message.

Errors

String Description
recog object not found Invalid recognizer object handle.
speech detector not found The recognizer object was created without a speech detector and no clipping statistics are available.
