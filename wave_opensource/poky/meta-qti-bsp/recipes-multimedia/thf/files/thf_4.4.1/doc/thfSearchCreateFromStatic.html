thfSearchCreateFromStatic

Syntax

#include <trulyhandsfree.h>
searchs_t *thfSearchCreateFromStatic(thf_t **thf, recog_t *r, sdata_t *global, 
                                     unsigned short index, 
                                     unsigned short nbest);

Description

Reads pre-compiled search from static memory and creates a search object. The pre-compiled search and the recognition object must have the same frame size. Call thfSearchDestroy to destroy the search object when no longer needed.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
sdata_t * global in Generic static object linked to static memory search object.
unsigned short index in Search index to static memory.
unsigned short nbest in Number of recognition results required

Return

Returns a unique handle to the search object (see searchs_t). Returns NULL in the event of error.

Errors

String Description
recog object not found Invalid recognizer object handle.
index is out of range The static object array contains less pronunC_t objects than expected.

See Also
