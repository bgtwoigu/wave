thfPronunCreateFromFile

Syntax

#include <trulyhandsfree.h>
pronuns_t *thfPronunCreateFromFile(thf_t **thf, const char *pronunfile, 
                                   unsigned short dialextIndex);

Description

Creates a pronunciation object that is used in generating N-best pronunciations for words/phrases. The pronunciation model is read from pronunfile. Pronunciation models are language-specific and are provided as part of the SDK. They often support a variety of dialects. The desired dialect must be specified by providing the corresponding dialect index. See language pack documentation for a list of supported dialects and indices. Call thfPronunDestroy to destroy the pronunciation object when no longer needed.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
const char * pronunfile in Pronunciation model file.
unsigned short dialextIndex in Dialect index.

Return

Returns a unique handle to the pronunciation object (see pronuns_t). Returns NULL in the event of an error.

Errors

String Description
Could not open raw file The datafile does not exist or the application had insufficient privileges to open the file for reading.
Error reading raw file: The datafile did not decode correctly. Was the file damaged?
Unlicensed pronunciation data Data object not properly licensed. Contact customer support.
Unknown pronunciation object type in file Data file could not be decoded. Probably an older or newer version.
