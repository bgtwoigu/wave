thfRecogGetSpeechRange

Syntax

#include <trulyhandsfree.h>
int thfRecogGetSpeechRange(thf_t **thf, recog_t *r, float *from, float *to);

Description

Identifies the start and end of speech as determined by the internal speech detector. The start time includes the backoff duration and end time include trailing silence. Parameters

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
float * from out Start of speech in milliseconds.
float * to out End of speech in milliseconds.

Return

Returns 1 to indicate success and 0 for an error. Use thfGetLastError to get error message.

Errors

String Description
recog object not found Invalid recognizer object handle.
speech detector not found The recognizer object was created without a speech detector and the speech range was thus not recorded.
