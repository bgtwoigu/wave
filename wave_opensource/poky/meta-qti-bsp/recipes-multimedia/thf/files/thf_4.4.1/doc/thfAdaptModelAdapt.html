thfAdaptModelAdapt

Syntax

#include <trulyhandsfree.h>
int thfAdaptModelAdapt(thf_t **thf, adaptObj_t *aObj, char *taskName, 
                       char *userName, float accuracy);

Description

Perform adaptation using enrollment data for the specified user(s), and create an adapted trigger (or set of triggers). The resulting adapted triggers are stored internally and can be obtained with a call to thfAdaptModelGetTrigger.

There is an input variable called 'accuracy'. This variable specifies whether a full adaptation is to be performed (yielding maximum accuracy but taking the longest amount of time), or a partial adaptation is to be performed (yielding lower accuracy but taking less time to adapt). The valid range for 'accuracy' is from 0.0 to 1.0. A value of 0.0 returns an adapted trigger the fastest, but will have a high rate of accepting noise and imposters. A value of 1.0 returns a trigger with full adaptation, yielding highest performance. There is a non-linear relationship between 'accuracy' and enrollment time, and also a non-linear relationship between 'accuracy' and performance. If trigger adaptation is too slow, one can perform two adaptations: the first with a low value of 'accuracy' (e.g. 0.10) to get a less accurate but more quickly available trigger, and the second (in the background) with 'accuracy' equal to 1.0 to get a more accurate trigger.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
adaptObj_t * aObj in adapt object.
char * taskName in name of current task.
char * userName in name of user for whom an adapted trigger is to be created, or NULL for all users in this task.
float accuracy in speed/accuracy tradeoff for adaptation.

Return

Returns 1 on success or 0 on error.

Errors

String Description
adapt object not found A NULL pointer was passed in for the adapt object; pass in a correctly-initialized adapt object pointer created by thfAdaptCreate.
adapt object not initialized A non-NULL pointer was passed in for the adapt object, but this pointer does not contain a properly-initialized adapt object that is the output of thfAdaptCreate.
task name not specified A valid task name must be provided; a task with this name must have been created with a previous call to thfAdaptTaskCreate.
