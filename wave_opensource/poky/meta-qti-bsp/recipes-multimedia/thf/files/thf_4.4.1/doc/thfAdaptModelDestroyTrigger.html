thfAdaptModelDestroyTrigger

Syntax

#include <trulyhandsfree.h>
int thfAdaptModelDestroyTrigger(thf_t **thf, thfModelTrigger_t *modelTrigger, 
                                int keepRecog, int keepSearch);

Description

Destroys a trigger object created with a call to thfAdaptModelGetTrigger.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
thfModelTrigger_t * modelTrigger in object containing an adapted trigger.
int keepRecog in 0 if recognizer object in this structure should be destroyed; else 1.
int keepSearch in 0 if search object in this structure should be destroyed; else 1.

Return

Returns 1 on success or 0 on error.

See Also

thfAdaptCreate, thfAdaptTaskCreate, thfAdaptUserAdd, thfAdaptEnrollmentAdd, thfAdaptModelAdapt and thfAdaptModelGetTrigger
