thfSpeakerGender

Syntax

#include <trulyhandsfree.h>
int thfSpeakerGender(thf_t **thf, recog_t *r, float *genderProb);

Description

Classify the utterance detected by thfRecogResult as belonging to a female or male speaker.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
float * genderProb out This value indicates the estimated gender of the speaker. If this value is less than 0.5, then the speaker is estimated to be male. If this value is greater than or equal to 0.5, then the speaker is estimated to be female.

Return

Returns 1 on success or 0 on error.

Errors

String Description
recog object not found Invalid recognizer object handle.
internal error: phrasespot search object not found Invalid search handle
internal error: search ROM struct not found Internal error: Contact customer support
speaker object not yet initialized Invalid speaker object handle
