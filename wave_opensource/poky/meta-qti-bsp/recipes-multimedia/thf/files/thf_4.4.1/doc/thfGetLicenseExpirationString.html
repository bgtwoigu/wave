thfGetLicenseExpirationString

Syntax

#include <trulyhandsfree.h>
const char *thfGetLicenseExpirationString(void);

Description

Returns the time and date when the current evaluation license will expire as a human-readable string, or "never" if the TrulyHandsfree SDK library was licensed with a perpetual, non-expiring key.

Parameters

None.

Return

A string of the format "%d %b %Y 00:00:00 GMT", e.g. "31 Dec 2017 00:00:00 GMT" or "never" if the library was licensed with a non-expiring key.

See Also

thfSessionCreate and thfGetLicenseExpiration
