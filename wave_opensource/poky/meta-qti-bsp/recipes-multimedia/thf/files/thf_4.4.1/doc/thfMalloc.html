thfMalloc

Syntax

#include <trulyhandsfree.h>
void *thfMalloc(size_t size);

Description

Allocates memory on the heap using the SDK allocator. Use thfFree to dispose of these.

This function is provided for convenience only.

A typical use scenario: An audio buffer can be allocated with thfWaveFromFile, or through some user-defined method. Instead of having to keep track of how the buffer was allocated (to ensure that the correct free function is called), the user-defined method can use thfMalloc. The buffer can then be released using thfFree, regardless of how it was initialized.

Parameters

Type Name Dir. Description
size_t size in Size in bytes to allocate

Return

A pointer to a newly allocated block of heap memory.
