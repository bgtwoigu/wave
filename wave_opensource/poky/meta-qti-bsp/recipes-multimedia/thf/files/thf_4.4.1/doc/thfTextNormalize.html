thfTextNormalize

Syntax

#include <trulyhandsfree.h>
const char *thfTextNormalize(thf_t **thf, pronuns_t *p, const char *text, 
                             unsigned short conditionCode);

Description

Performs text normalization on the input string identifying and transforming things like abbreviations, acronyms, numbers, currency, dates and times into full text as needed. For example '$5' becomes 'five dollars'. A condition code is used to specify which normalization rules get applied. Condition codes are language specific. See corresponding language pack documentation for a list of supported condition codes. They can combined in various combinations using the OR operator as needed.

NOTE: Text normalization is not performed on the following special strings: "*nota", "*any", "*garbage" and "*sil".

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
pronuns_t * p in Pronunciation object.
const char * text in Text to be normalized.
unsigned short conditionCode in Condition code.

Return

Returns a normalized text string. Use thfFree to free returned string memory when you're done. Returns NULL in the event of an error.

Errors

String Description
Unlicensed pronunciation data Original data file not properly licensed. Contact customer support.
