thfPronunCreateFromStatic

Syntax

#include <trulyhandsfree.h>
pronuns_t *thfPronunCreateFromStatic(thf_t **thf, sdata_t *global, 
                                     unsigned short index, 
                                     unsigned short dialextIndex);

Description

Creates a pronunciation object that is used for generating N-best pronunciations for words/phrases. The pronunciation model is read from static memory, as specified by the index argument. Pronunciation models are language-specific and are provided as part of the SDK. They often support a variety of dialects. The desired dialect must be specified by providing the corresponding dialect index. See language pack documentation for a list of supported dialects and indices. Call thfPronunDestroy to destroy the pronunciation object when no longer needed.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
sdata_t * global in Generic static object array, linked to static memory pronunciation model.
unsigned short index in Index into global of the specific static object to initialize from.
unsigned short dialextIndex in Dialect index.

Return

Returns a unique handle to the pronunciation object (see pronuns_t). Returns NULL in the event of an error.

Errors

String Description
index is out of range The static object array contains less pronunC_t objects than expected.
Unlicensed pronunciation data Original data was not properly licensed. Contact customer support.
