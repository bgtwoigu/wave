thfTtsPrep

Syntax

#include <trulyhandsfree.h>
unsigned long thfTtsPrep(thf_t **thf, tts_t *tts, const char *text, 
                         float pitch, float speechrate, unsigned long *frameSz);

Description

Prepares Tts engine for generating pipelined speech. This means audio will be synthesized as a series of frames (i.e., chunks). This is an alternative to thfSynthesize which generates a single large audio buffer for the entire text. Control speech characteristics using the pitch parameter (legal range 0.5 to 2.0) and speech rate parameter (legal range 0.5 to 1.25). Use thfTtsGetFrame repeatedly to retrieve the synthesized audio buffers.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
tts_t * tts in Tts object.
const char * text in Text
float pitch in Pitch parameter.
float speechrate in Speech rate parameter.
unsigned long * frameSz out Number of samples in a frame.

Return

Returns the number of frames to be synthesized. Returns 0 in the event of an error.

Errors

String Description
No tts object No tts object handle provided. Use thfTtsCreateFromFile or thfTtsCreateFromStatic to create tts object handle.
getState failed Internal error. Contact customer support.
