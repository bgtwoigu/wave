thfRecogDowngradeToCnn

Syntax

#include <trulyhandsfree.h>
int thfRecogDowngradeToCnn(thf_t **thf, recog_t *r);

Description

This function will, if possible, downgrade a recognizer's network from DNN to CNN. If it is not possible, the function will return 0 as an error.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.

Return

Returns 1 on success or 0 on error.

Errors

String Description
recog object not found Invalid recognizer object handle
recog ROM structure not found Recognizer object handle exists, but it likely invalid
no DNN available to downgrade The recognizer exists, but does not contain a DNN to downgrade.
failed to downgrade net The conversion from DNN to CNN failed. The most likely reason is that the DNN can not be convered to the CNN format.
