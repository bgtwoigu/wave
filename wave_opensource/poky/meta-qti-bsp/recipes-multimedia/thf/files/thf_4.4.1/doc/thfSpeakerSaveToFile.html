thfSpeakerSaveToFile

Syntax

#include <trulyhandsfree.h>
int thfSpeakerSaveToFile(thf_t **thf, recog_t *r, unsigned short sID, 
                         char *filename);

Description

Save the recordings and/or model associated with a speaker to file. The recordings are those stored internally by thfSpeakerStoreLastRecording, and the model is that created by thfSpeakerAdapt.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
unsigned short sID in The identification (integer value) of the speaker whose data and/or model are to be stored.
char * filename in Name of file that data and/or model gets saved into. PARARM unsigned short set in A flag to indicate what kind of information should be stored. Valid values are SPEAKER_SAVE_DATA (to store all of the recordings available for this speaker), SPEAKER_SAVE_MODEL (to store the model of this speaker generated by thfSpeakerAdapt and used for speaker verification or speaker ID), or SPEAKER_SAVE_BOTH (to store both the data and the model).

Return

Returns 1 on success or 0 on error.

Errors

String Description
filename passed to thfSpeakerSaveToFile is NULL Invalid filename parameter
recog object not found Invalid recognizer object handle.
speaker object not yet initialized Invalid speaker object handle
can't find speaker ID Invalid speaker ID parameter
internal error: found speaker UBM but not speaker acoustic model Internal error: contact customer support
failed to write data to file Failed to write data
