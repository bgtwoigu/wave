thfCheckRecording

Syntax

#include <trulyhandsfree.h>
int thfCheckRecording(thf_t **thf, recog_t *r, short *wave, 
                      unsigned long waveLen, unsigned long *feedback);

Description

Performs simple evaluation of the quality of a waveform that is assumed to contain speech. The evaluation result is put in the output variable 'feedback'. Specific bits of 'feedback' are set to 1 to indicate a problem; therefore, if 'feedback' is 0, the recording passed the evaluation check with no detected problems. The definitions of specific bits are specified in trulyhandsfree.h, in #define statements of the form CHECKRECORDING_BITFIELD_*. The meanings of the bitfields are given in the list of recording quality parameters.

Quality measure thresholds have reasonable default values, but can be modified using thfRecogConfigSet.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object that will be used to process the recording.
short * wave in waveform, expressed as an array of short integers.
unsigned long waveLen in length of waveform 'wave'.
unsigned long * feedback out evaluation results.

Return

Returns 1 on success or 0 on error. In case of error, feedback output is set to 0xFFFF.

Errors

String Description
recog object not found Invalid recognizer object handle.
internal error: recog ROM struct not found Unexpected internal error. Contact customer support.
