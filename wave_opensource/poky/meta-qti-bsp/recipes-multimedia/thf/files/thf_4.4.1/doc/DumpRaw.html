DumpRaw

Syntax

TrulyHandsfree SDK\<platform>\bin\dumpRaw filebasename

Description

Converts compiled searches and supplied data files to C-language structures, suitable for compilation into an application. The application loads the data file:filebasename.raw and stores the resulting source code in filebasename.c.

Parameters

Name Dir. Description
filebasename in The base filename used to load the binary .raw file and save the output .c source file.
