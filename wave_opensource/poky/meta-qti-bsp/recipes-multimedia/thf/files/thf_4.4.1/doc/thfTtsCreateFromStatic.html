thfTtsCreateFromStatic

Syntax

#include <trulyhandsfree.h>
tts_t *thfTtsCreateFromStatic(thf_t **thf, sdata_t *global, 
                              unsigned short index);

Description

Reads TTS voice data from static memory and creates a TTS object. Call thfTtsDestroy to destroy the TTS object when no longer needed.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
sdata_t * global in Generic static object array, linked to static memory containing tts data.
unsigned short index in Index into global of the specific static object to initialize from.

Return

nothing.

Errors

String Description
index is out of range The static object array contains an incorrect number of tts_t components.
Unlicensed TTS data Data object not properly licensed. Contact customer support.

See Also

Compiling data into application, thfFree and thfTtsDestroy
