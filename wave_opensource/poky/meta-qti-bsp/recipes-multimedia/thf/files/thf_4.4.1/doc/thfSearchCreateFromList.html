thfSearchCreateFromList

Syntax

#include <trulyhandsfree.h>
searchs_t *thfSearchCreateFromList(thf_t **thf, recog_t *r, pronuns_t *p, 
                                   const char **words, const char **pronuns, 
                                   unsigned short count, unsigned short nbest);

Description

Creates a search object for the specified list of words. Pronunciation can be either supplied or generated automatically. If pronunciations are to be generated, set parameter 'pronuns' to NULL. Multiple word entries are concatenated with the '_' underscore symbol in the compiled search. The search object can be used immediately or saved to file for later use. Call thfSearchDestroy to destroy the search object when no longer needed. You must use thfSearchCreateFromGrammar if you are using a task-specific acoustic model.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
pronuns_t * p in Pronunciation object.
const char ** words in Array of words to build.
const char ** pronuns in Array of word pronunciations. Can be NULL if pronunciations are to be generated internally to SDK. (see How to generate pronunciations).
unsigned short count in Number of words (and pronunciations if provided) in array.
unsigned short nbest in Number of recognition results required

Return

Returns a unique handle to the search object (see searchs_t). Returns NULL in the event of error.
