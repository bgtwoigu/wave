thfTtsGetFrame

Syntax

#include <trulyhandsfree.h>
short *thfTtsGetFrame(thf_t **thf, tts_t *tts, unsigned long *speechLen);

Description

Retrieve a frame of audio.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
tts_t * tts in Tts object.
unsigned long * speechLen out Number of speech samples synthesized.

Return

Returns a pointer to a frame of 16-bit linear PCM synthesized audio. Returns NULL in the event of an error. . Use thfFree to free returned speech buffer when you're done.

Errors

String Description
No tts object No tts object handle provided. Use thfTtsCreateFromFile or thfTtsCreateFromStatic to create tts object handle.
uninitialized tts object Uninitialized object. Must call thfTtsPrep to initialized Tts object.
nothing to synthesize Nothing to synthesize or thfTtsPrep returned an error.
no more frames available All audio has been returned already.

See Also

thfFree, thfTtsCreateFromFile and thfTtsCreateFromStatic
