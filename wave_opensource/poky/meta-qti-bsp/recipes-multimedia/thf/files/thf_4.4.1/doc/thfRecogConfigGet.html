thfRecogConfigGet

Syntax

#include <trulyhandsfree.h>
int thfRecogConfigGet(thf_t **thf, recog_t *r, thfRecogConfig_t key, 
                      float *value);

Description

Used to query various recognition parameters. The parameter is identified by a key and the value is returned as a float.

Valid keys are in the enumerated type thfRecogConfig_t.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object
recog_t * r in Recognizer object
thfRecogConfig_t key in Identifies parameter to get
float * value out Retrieved parameter value

Return

Returns 1 to indicate success and 0 for an error. Use thfGetLastError to get error message.

Errors

String Description
unknown thfRecogConfig_t key They key parameter, used to identify the recognition parameter to retrieve, is not valid.
