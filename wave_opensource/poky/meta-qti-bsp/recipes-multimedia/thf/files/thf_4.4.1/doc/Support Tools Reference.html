Support Tools Reference

The Support Tools Reference section provides information about how to use the TrulyHandsfree SDK utilities:
  • CreatePronun: A utility to create an ordered list of pronunciations from an input file of phrases, suitable for inclusion into C-language code.
  • DumpRaw: A utility to convert compiled searches and supplied data files to C-language structures, suitable for compilation into an application.
  • deepdump: A command-line utility to convert an acoustic model and search file into deeply embedded format.
  • enroll: A command-line utility to create search and acoustic models for user-defined triggers (UDT) and enrolled fixed triggers (EFT), with or without additional speaker identification/verification.
  • pseval: A command-line utility to run a phrase spotter over an audio file.
  • recogeval: A command-line utility to run a recognizer and speech detector over an audio file.
  • sdet: A command-line utility to take in a wave file and output the detected speech to another.
  • svenroll: Deprecated: Speaker verification fixed-phrase enrollment: Creates compiled search and acoustic model files from example fixed-phrase recordings for each user. This utility has been superseded by 'enroll'.
  • udtenroll: A deprecated command-line utility to create search and acoustic models for user-defined triggers and speaker identification/verification. This utility has been superseded by 'enroll'.
