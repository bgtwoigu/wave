thfGetLicenseExpiration

Syntax

#include <trulyhandsfree.h>
time_t thfGetLicenseExpiration(void);

Description

Returns the time and date when the current evaluation license will expire, or 0 if the TrulyHandsfree SDK library was licensed with a perpetual, non-expiring key.

Parameters

None.

Return

The timestamp (in seconds since the Unix epoch) when the SDK evaluation license will expire, or 0 if using a perpetual license.

See Also

thfSessionCreate and thfGetLicenseExpirationString
