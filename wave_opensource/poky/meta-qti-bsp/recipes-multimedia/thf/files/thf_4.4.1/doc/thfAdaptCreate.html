thfAdaptCreate

Syntax

#include <trulyhandsfree.h>
adaptObj_t *thfAdaptCreate(thf_t **thf);

Description

Creates an 'adapt' object to be used in the creation of user-defined triggers and/or enrolled fixed triggers. The created object is empty, and will need to have task-specific data added to it with thfAdaptTaskCreate. Multiple tasks (e.g. a user-defined trigger and an enrolled fixed trigger) can be included in the same adapt object.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.

Return

Returns a pointer to an 'adapt' object on success, or NULL on error.

Errors

String Description
Error creating adapt object Unexpected internal error. Contact customer support.

See Also

thfAdaptDestroy and thfAdaptTaskCreate
