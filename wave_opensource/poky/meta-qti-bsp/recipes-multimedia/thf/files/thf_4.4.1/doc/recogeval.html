recogeval

Syntax

TrulyHandsfree SDK\<platform>\bin\recogeval [options] -n netfile -s searchfile wavefile

Description

An evaluation tool for a non-phrasespot recognizer. Given an acoustic model, search file and an audio file, print to stdout a time-stamped list of all occurrences of recognized phrases. Does optional speech detection and display of word and phoneme alignments, and can save recognized portions of the audio file.

Parameters

Name Default Description
-n netfile Neural network (acoustic model) file name.
-s searchfile Compiled phrase-spotting search file name.
wavefile Audio file. This can be in RIFF (.wav) format with 16-bit linear PCM encoding, or a headerless 16-bit linear PCM file in host byte order. If the latter, the -r rate flag is required to specify the sample rate.
-g wavefilepath Sets the recognized wave save file.
-a any .5 Sets the any search parameter.
-N nota .5 Sets the nota search parameter.
-B beam 250.0 Sets the beam search parameter.
-t ts 400.0 Sets the trailing silence recognizer parameter.
-r rate read from wavfile Audio sample rate in Hz, required if the wave file is in PCM format.
-d disabled Disables speech detection.
-p disabled Show phoneme alignments.
-w disabled Show word alignments.
-v disabled Produce verbose progress messages.
