thfAdaptModelGetTrigger

Syntax

#include <trulyhandsfree.h>
int thfAdaptModelGetTrigger(thf_t **thf, adaptObj_t *aObj, 
                            char *taskAndUserListIn, 
                            thfModelTrigger_t **modelTriggerOut);

Description

Return an object containing adapted triggers that were created with a call to thfAdaptModelAdapt. The object has the structure described in thfModelTrigger_t and includes the number of users in the resulting trigger, and pointers to the adapted recognizer and search. Results from multiple tasks and users can be combined into a single trigger.

The list of tasks and users is specified with a character array containing the task name(s) and the users within each task listed within curly braces. For example, to obtain a combined trigger for all users in the task 'SensoryTrigger', the list is the single character string 'SensoryTrigger'. To obtain a trigger for the user 'Alice' in the task 'SensoryTrigger', the list is 'SensoryTrigger {Alice}'. To obtain a trigger for the users Alice and Bob in the task 'SensoryTrigger' and the user 'Charlie' in the task 'UserTrigger', the list is 'SensoryTrigger {Alice Bob} UserTrigger {Charlie}'. Note that this list is case sensitive.

If no users are specified and no adaptation has been performed, the result is the unadapted trigger. For an enrolled fixed trigger, this includes the acoustic model and the search. For a user-defined trigger, this includes only the acoustic model; the search field will be set to NULL.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
adaptObj_t * aObj in adapt object.
char * taskAndUserListIn in a list of the tasks and users for which adapted triggers are to be obtained. The format for this list is described above.
thfModelTrigger_t ** modelTriggerOut out object containing the combined and adapted trigger(s).

Return

Returns 1 on success or 0 on error.

Errors

String Description
adapt object not found A NULL pointer was passed in for the adapt object; pass in a correctly-initialized adapt object pointer created by thfAdaptCreate.
adapt object not initialized A non-NULL pointer was passed in for the adapt object, but this pointer does not contain a properly-initialized adapt object that is the output of thfAdaptCreate.
task/user list not specified A NULL pointer was passed in for the list of tasks and users. This list must be an ASCII string with the format described above.
