thfFree

Syntax

#include <trulyhandsfree.h>
void thfFree(void *p);

Description

Frees memory that was allocated internally by one of the SDK functions. Certain documented functions return data structures that must be deallocated when no longer needed.

Parameters

Type Name Dir. Description
void * p in A pointer to memory previously allocated by another SDK function.

Return

None.
