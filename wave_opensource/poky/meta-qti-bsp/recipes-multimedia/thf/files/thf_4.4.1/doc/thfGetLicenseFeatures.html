thfGetLicenseFeatures

Syntax

#include <trulyhandsfree.h>
const char *thfGetLicenseFeatures(void);

Description

Returns an encoded string that describes the modules and features that are enabled in this SDK library. This string is not human-readable.

Do not use this function unless requested by Sensory Technical Support.

Parameters

None.

Return

An encoded nul-terminated ASCII string.

See Also

thfSessionCreate, thfGetLicenseExpiration and thfGetLicenseExpirationString
