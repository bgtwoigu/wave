thfTtsSynthesize

Syntax

#include <trulyhandsfree.h>
short *thfTtsSynthesize(thf_t **thf, tts_t *tts, const char *text, float pitch, 
                        float speechrate, unsigned long *speechLen);

Description

Generates synthesized speech for the input text. Produces a single large audio block as opposed to thfTtsPrep which synthesizes audio in smaller blocks. Control speech characteristics using the pitch parameter (legal range 0.5 to 2.0) and speech rate parameter (legal range 0.5 to 1.25). Use thfFree to free returned speech buffer when you're done.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
tts_t * tts in Tts object.
const char * text in Text
float pitch in Pitch parameter.
float speechrate in Speech rate parameter.
unsigned long * speechLen out Number of speech samples synthesized

Return

Returns a pointer to a block of 16-bit linear PCM synthesized audio. Returns NULL in the event of an error.

Errors

String Description
No tts object No tts object handle provided. Use thfTtsCreateFromFile or thfTtsCreateFromStatic to create tts object handle.
getState failed Internal error. Contact customer support.

See Also
