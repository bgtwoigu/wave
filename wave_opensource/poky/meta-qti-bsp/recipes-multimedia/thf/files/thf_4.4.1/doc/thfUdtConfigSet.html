thfUdtConfigSet

Syntax

#include <trulyhandsfree.h>
int thfUdtConfigSet(thf_t **thf, udt_t *udt, thfUdtConfig_t key, float value);

Description

Set one of the parameters used in UDT enrollment. Valid keys to use for parameters, and their meanings, are in the enumerated type thfUdtConfig_t. The default values, set by thfUdtCreate, are reasonable values; thfUdtConfigSet() does not normally need to be called.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
udt_t * udt in Udt object.
thfUdtConfig_t key in specification of which parameter to set
float value in new parameter value

Return

Returns 1 on success or 0 on error.

Errors

String Description
udt object not found Invalid udt object handle.
can't set num param A to zero 'key' specifies the number of 'param A' values, which must be at least 1.
can't set num param B to zero 'key' specifies the number of 'param B' values, which must be at least 1.
value must be in 'ushort' range 'key' specifies the maximum value of 'param D' or 'delay', which must be within the range of an unsigned short.
can't set num chunks to zero 'key' specifies the number of chunks to use in tuning, which must be at least 1.
value must be in 'long' range 'key' specifies the maximum amount of memory to use, which must be greater than or equal to zero.
value must be 0 or 1 'key' specifies whether or not silence is required around each UDT phrase, and the value must be 0 (no silence required) or 1 (silence is required).
unknown key The 'key' parameter, used to identify the UDT parameter to retrieve, is not valid.
