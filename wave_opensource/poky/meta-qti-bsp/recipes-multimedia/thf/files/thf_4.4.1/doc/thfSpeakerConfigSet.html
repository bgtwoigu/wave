thfSpeakerConfigSet

Syntax

#include <trulyhandsfree.h>
int thfSpeakerConfigSet(thf_t **thf, recog_t *recog, unsigned short sID, 
                        thfSpeakerConfig_t key, float value);

Description

Used to set a speaker verification or speaker ID parameter of the recognition object. A parameter is set by passing a key to indicate which parameter and a value that the parameter will be set to. Values are passed in using floating-point numbers. Only one parameter can be set per call to this function.

Valid keys to use for parameters, and their meanings, are in the enumerated type thfSpeakerConfig_t. The default values, set by thfSpeakerInit, are reasonable starting points, but thfSpeakerConfigSet() may need to be called since different triggers may need somewhat different settings for optimal performance. Not all parameters can be set; some keys can only be used to query parameter values.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * recog in Recognizer object.
unsigned short sID in The identification (integer value) of the speaker whose parameters are to be set. For SPEAKER_TRIGGER_SAMP_PER_CAT, SPEAKER_SAMP_PER_CAT_WITHIN, SPEAKER_TARGET_SNR, SPEAKER_DUR_MIN_FACTOR, SPEAKER_DUR_MAX_FACTOR, and SPEAKER_DSP_SV_THRESH this value is ignored and the parameter is set for all speakers.
thfSpeakerConfig_t key in specification of which parameter to set
float value in new parameter value

Return

Returns 1 on success or 0 on error.

Errors

String Description
recog object not found Invalid recognizer object handle.
speaker object not yet initialized Invalid speaker object handle.
unknown set option The value of setName does not indicate a valid parameter.
can't find subject ID Invalid speaker ID parameter
no adapted acoustic model exists If setting SPEAKER_RECOG_WITH_ADAPTED to a non-zero value, there is no existing acoustic model for this user.
speedAccuracy must be from 1 to 100 A value less than 1 or greater than 100 was used to set SPEAKER_SPEED_ACCURACY. Such a value is not valid for this parameter.
