thfRecogCombine

Syntax

#include <trulyhandsfree.h>
int thfRecogCombine(thf_t **thf, int prune, recog_t *r1, recog_t *r2, 
                    searchs_t *s1, searchs_t *s2, const char *prefix1, 
                    const char *prefix2, recog_t **outputRecog, 
                    searchs_t **outputSearch, float *compressionRation);

Description

Combines two different recognizers (acoustic models and searches) into a single recognizer (with one acoustic model and one search.)

Acoustic models (recognizer objects) that were derived from the same base model can be combined efficiently. UDT models adapted from the same acoustic model is a prime example of this.

The compressionRatio return value is the ratio of the size of the combined acoustic model to the sum of the sizes of the source models. For efficiently combined models this is typically around 0.7. Where models do not share enough parameters to allow efficient combination, this will typically be around 1.4 (i.e. 40% larger than the sum of the source models.) Memory and computational requirements scale linearly with the size of the acoustic model.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
int prune in Prune acoustic models if non-zero.
recog_t * r1 in First recognizer object to be combined or NULL.
recog_t * r2 in Second recognizer object to be combined.
searchs_t * s1 in First search object to be combined or NULL.
searchs_t * s2 in Second search object to be combined.

searchs_t ** outputSearch out Combined search object.
float * compressionRation out Acoustic model compression ratio estimate.

Return

Returns 1 on success or 0 on error.
