thfSpeakerReadFromFile

Syntax

#include <trulyhandsfree.h>
int thfSpeakerReadFromFile(thf_t **thf, recog_t *r, unsigned short sID, 
                           char *filename, unsigned short flag, 
                           unsigned short force);

Description

Read the recordings and/or model associated with a speaker from a file. These recordings or model must have been written to file using thfSpeakerSaveToFile.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
unsigned short sID in The identification (integer value) of the speaker to whom the recordings and/or model will be assigned.
char * filename in Name of file that data and/or model are read from.
unsigned short flag in A flag to indicate what kind of information should be obtained from the file. Valid values are SPEAKER_READ_DATA (to read only the data), SPEAKER_READ_MODEL (to read only the model), and SPEAKER_READ_BOTH (to read both data and model).
unsigned short force in A flag to indicate what to do if the speaker ID specified in 'sID' already exists in the recognizer object. If force is 0 and the speaker ID already exists, then an error is returned. If force is 1 and the speaker ID already exists, then any model or data associated with the speaker ID is removed, and the new data and/or model is assigned to this speaker ID. Setting 'force' to 0 can sometimes be useful in debugging.

Return

Returns 1 on success or 0 on error.

Errors

String Description
recog object not found Invalid recognizer object handle.
filename passed to thfSpeakerReadFromFile is NULL Invalid filename parameter
unable to open speaker file File open failed
speaker object not yet initialized Invalid speaker object handle
speaker ID already exists in speaker object The value of 'force' is 0 and the recognizer object already contains the speaker ID 'sID'.
