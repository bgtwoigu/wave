thfRecogCreateFromStatic

Syntax

#include <trulyhandsfree.h>
recog_t *thfRecogCreateFromStatic(thf_t **thf, sdata_t *global, 
                                  unsigned short index, unsigned long maxbuf, 
                                  unsigned short backoff, char useSdet);

Description

Reads in the acoustic model from static memory and creates a recognizer object. Memory buffers are allocated internally for the audio and backoff buffers. The size of the audio buffer will depend on your application needs. Set the backoff parameter to 0xffff to use the default value. If no speech detector is required then the backoff parameter is ignored. Call thfRecogDestroy to destroy the recognizer object when no longer needed. Parameters

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
sdata_t * global in Generic static object linked to static memory recognizer object.
unsigned short index in Index of acoustic model into global.
unsigned long maxbuf in Maximum size of audio buffer (in samples) to be fed to recognizer.
unsigned short backoff in Size of speech detector backoff buffer (in milliseconds). Set to 0xffff for default values.
char useSdet in Binary flag indicates whether to create an speech detector for use during recognition (see Creating a Recognizer).

Return

Returns a unique handle to the recognizer object (see recog_t). Returns NULL in the event of error.

Errors

String Description
index is out of range The static object array contains less recogC_t objects than expected.
No net found in static data structure The static data structure appears to be a valid SDK structure, but did not contain any acoustic model data.
Incompatible net found in static data structure The acoustic model static data is based on an unknown feature extraction module.
Error initializing net Could not allocate scratch memory
Error starting DSP Node Could not start feature extraction module (applicable only on some multiprocessor platforms).
Could not start async thread Could not start feature extraction module (applicable only on some multithreaded platforms).
