thfRecogCreateHybrid

Syntax

#include <trulyhandsfree.h>
int thfRecogCreateHybrid(thf_t **thf, recog_t **r, pronuns_t **p, 
                         unsigned short numObjects, const char ***words, 
                         const char ***pronuns, unsigned short *numWords, 
                         recog_t **hybridRecog, pronuns_t **hybridPronun);

Description

Creates a hybrid recognizer from a set of individual recognition objects. Each supplied recognition object must have a corresponding pronunciation object, a list of words and a list of pronunciations. The call returns handles to a hybrid recognition and hybrid pronunciation object which can be used as normal with the standard SDK calls. The resulting hybrid recognizer object must be initialized before it can be used (see thfRecogInit). Words and pronunciations are owned by the caller and should be kept around until a search object has been generated (e.g., thfCreateSearchFromGrammar). When including a task-specific recognizer (e.g., digits only) the corresponding pronunciation list must be set to NULL. When using the hybrid recognizer with a pre-built search (e.g., thfSearchCreateFromFile or thfSearchCreateFromStatic) arguments 3, 5, 6, 7 and 9 must be set to NULL. Call thfRecogDestroy and thfPronunDestroy to destroy the hybrid recognition and hybrid pronunciation objects when no longer needed. All dependent recognition and pronunciation objects will also get destroyed at that time.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t ** r in Array of recognizer objects.
pronuns_t ** p in Array of pronunciation objects. Use NULL for a pre-built search.
unsigned short numObjects in Number of recognizer/pronunciation objects.
const char *** words in Array of array of words in grammar. Use NULL for a pre-built search.
const char *** pronuns in Array of array of pronunciations in grammar. Use NULL for a pre-built search.
unsigned short * numWords in Array of number of words/pronunciations. Use NULL for a pre-built search. Use NULL task-specific recognizer array elements.
recog_t ** hybridRecog out Handle to hybrid recognition object created.
pronuns_t ** hybridPronun out Handle to hybrid pronuncation object created. Use NULL for a pre-built search.

Return

Returns 1 on success or 0 on error.

Errors

String Description
Hybrid count must be greater than one 'numObjects' parameter too small.
Recog object list not specified Array of recognition objects 'r' cannot be NULL.
Pronun object list not specified Array of pronuncation objects 'p' cannot be NULL.
