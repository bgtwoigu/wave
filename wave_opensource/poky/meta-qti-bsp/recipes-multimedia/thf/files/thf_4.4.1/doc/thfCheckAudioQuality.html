thfCheckAudioQuality

Syntax

#include <trulyhandsfree.h>
int thfCheckAudioQuality(thf_t **thf, recog_t *r, short*wav, 
                         unsigned long wavLen);

Description

Performs audio quality checks, including checks for clipping and saturation.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
short* wav to a valid wave file.
unsigned long wavLen indicating the length of the wave file..

Return

Returns 1 if valid audio or 0 on poor or bad quality audio.

See Also

thfRecogCreateFromFile, thfRecogCreateFromFunc and thfRecogCreateFromStatic
