thfTtsConfigGet

Syntax

#include <trulyhandsfree.h>
int thfTtsConfigGet(thf_t **thf, tts_t *tts, thfTtsConfig_t key, int *value);

Description

Used to read values of various TTS parameters that change the default synthesis behavior.

Valid keys to use for parameters are in the enumerated type thfTtsConfig_t.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
tts_t * tts in Tts object.
thfTtsConfig_t key in Parameter name.
int * value out Current value for parameter selected by key.

Return

Returns 1 on success or 0 on error.

Errors

String Description
no tts object tts argument is not a valid Tts object.
configuration key is out of range key is not one of the values in the thfTtsConfig_t enum.
