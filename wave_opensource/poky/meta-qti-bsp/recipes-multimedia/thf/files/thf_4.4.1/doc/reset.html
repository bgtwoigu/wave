Re-Using the Recognizer

The recognizer can be used multiple times, but it must be reset between each use, by calling thfRecogReset. Even if no results were available, if you've called thfRecogPipe you must reset it before starting over.

Alternatively, if you need to change the recognition vocabulary or underlying acoustic model, call thfRecogInit with the relevant recog and search objects.
