thfUdtCacheSaveToFile

Syntax

#include <trulyhandsfree.h>
int thfUdtCacheSaveToFile(thf_t **thf, udt_t *udt, const char *cacheFilename);

Description

Write the set of enrollment data for all users in the udt object to file. This function must be used in careful conjunction with the data from the thfUser_t object. In particular, thfUdtCacheSaveToFile saves the enrollment data, but not the user name, which must be saved separately. Enrollment data are stored in the udt object upon calling thfUdtCheckEnrollments.

The data that are saved to file can be read in later using a call to thfUdtCacheLoadFromFile.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
udt_t * udt in Udt object.
const char * cacheFilename in character string containing the name of the file to write data to.

Return

Returns 1 on success or 0 on error.

Errors

String Description
UDT object not found Invalid udt object handle.
Number of users cannot be zero At least one user must have data stored in the udt object. User data are stored when calling thfUdtCheckEnrollments.
can't find userCache container The userCache container type can't be found, probably indicating an SDK version conflict."
can't open file for writing The output file could not be written to. The file system denied access for writing to the datafile.
